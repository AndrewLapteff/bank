# Simple Bank Project

##### Frontend

- React
- MobX
- React Query
- Axios
- ChartJS

##### Backend

- NestJS
- Prisma
- JWT
- Bcrypt

##### SQL

- PostgreSQL

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
