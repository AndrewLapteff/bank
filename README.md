# Simple Bank Project

##### Frontend

- React
- MobX

##### Backend

- NestJS
- Prisma
- JWT
- Bcrypt

##### SQL

- PostgreSQL

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
